# ⚾ KBOSPI

KBO 10개 구단을 대상으로 한 **모의 주식 투자 게임**입니다.
실제 KBO 경기 결과에 따라 구단 주가가 변동되며, 가상의 자금으로 매수·매도를 즐길 수 있습니다.

> 🚧 **BETA** — 현재 개발 진행 중입니다.

🔗 **[KBOSPI Preview](https://chihyuns0ng.github.io/KBOSPI/)**

---

## 주요 기능

### 📅 경기 일정·결과
- 월별 KBO 경기 일정 및 결과 조회
- 구단별 필터링
- 팀 순위표 (승/패/무/승률/게임차)

### 📈 KBOSPI
- KBO 10개 구단 실시간 주가 조회
- 구단별 주가 차트 및 등락률 표시
- 가상 자금(초기 1,000만원)으로 매수·매도
- 거래 가능 시간 제한 (00:30 ~ 14:00 KST)
- 00:00 ~ 00:30 경기 결과 자동 반영 구간 (거래 불가)
- 참가자 수익률 순위 리더보드

### 💬 자유게시판
- 자유 / 투자 토론 / 분석 카테고리
- 로그인 없이 읽기 가능
- 로그인 후 글쓰기·댓글·좋아요

### 📮 문의사항
- 로그인 없이 목록 열람 가능
- 로그인 후 문의 작성 (공개/비공개 선택)
- 관리자 답변 기능

### 🔐 회원
- 아이디 중복확인 + 비밀번호 회원가입
- 로그인 / 로그아웃

### 🛠 관리자
- 경기 결과 수동 입력 (주가 즉시 반영)
- 회원 목록 및 자산 현황 조회
- 게시글 관리 (공지 고정/해제, 삭제)
- 공지사항 작성
- 주가 / 자금 / 전체 게임 초기화

## 시작 주가 기준

> ⚠️ 아래 수치는 예시 자료입니다. 실제 서비스에서는 KBO 리그 실시간 순위를 반영한 시작 주가 산정을 목표로 합니다.

| 순위 | 구단 | 시작 주가 (예시) |
|---|---|---|
| 1위 | KT 위즈 | 150,000원 |
| 2위 | LG 트윈스 | 140,000원 |
| 3위 | SSG 랜더스 | 130,000원 |
| 4위 | 삼성 라이온즈 | 120,000원 |
| 5위 | KIA 타이거즈 | 110,000원 |
| 6위 | NC 다이노스 | 100,000원 |
| 공동 7위 | 한화 이글스 / 두산 베어스 | 90,000원 |
| 9위 | 키움 히어로즈 | 70,000원 |
| 10위 | 롯데 자이언츠 | 60,000원 |

## 운영 시간

| 시간 (KST) | 상태 |
|---|---|
| 00:00 ~ 00:30 | ⚙️ 경기 결과 반영 중 (거래 불가) |
| 00:30 ~ 14:00 | ✅ 거래 가능 |
| 14:00 ~ 24:00 | 🔒 거래 마감 (경기 시간대) |

---

## 주가 변동 방식

> ⚠️ 현재는 단순 랜덤 변동 방식을 사용하고 있습니다. 향후 타율, 방어율, 득점차 등 실제 경기력 지표를 반영한 계산식 정립을 목표로 합니다.

| 결과 | 현재 변동폭 |
|---|---|
| 승 | +5 ~ +8% (랜덤) |
| 패 | -4 ~ -7% (랜덤) |
| 무 | ±1% (랜덤) |

## 기술 스택

### 현재 (프론트엔드 단독)
- HTML / CSS / JavaScript (Vanilla)
- Chart.js
- localStorage 기반 데이터 저장

### 예정 (서버 연동)
- **Backend**: Node.js + Express
- **Database**: PostgreSQL
- **Hosting**: Render
- **크롤링**: 네이버 스포츠 (경기 결과 자동 수집)
- **스케줄러**: node-cron (매일 자정 자동 실행)
